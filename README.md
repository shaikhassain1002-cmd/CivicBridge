## Problem

In many cities, citizens face civic problems like potholes, garbage dumping, broken street lights, and water leaks.
There is no simple platform for people to quickly report these issues to the authorities.

## Solution

CivicBridge is a web platform that allows citizens to report civic issues easily.
Users can submit complaints with descriptions, images, and location details so that authorities can track and resolve the issues efficiently.

## Features

* Report civic problems
* Upload images of the issue
* Track complaint status
* Community interaction
* Simple and user-friendly interface

## Tech Stack

Frontend

* HTML
* CSS
* JavaScript

Backend

* Node.js

Database

* Fire Base

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/shaikhassain1002-cmd/CivicBridge.git
```

2. Go to the project folder

```
cd CivicBridge
```

3. Install dependencies

```
npm install
```

4. Run the server

```
npm start
```

5. Open in browser

```
http://localhost:3000
```
